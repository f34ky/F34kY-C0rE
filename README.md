# F34kY-C0rE
F34kY Core is a lightweight Python utility for Roblox and other games. Features include BunnyHop (auto‑jump), Auto Clicker, CS:GO‑style crosshair, Clock, FPS and Ping counters. Fully customizable colors and hotkeys. Borderless draggable window, config saving, and easy‑to‑use interface. Open source under MIT license. 🎮
